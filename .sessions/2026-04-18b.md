## What Was Built

- `css/style.css` — Full dark theme. Body `#0d1117`, canvas bordered, control panel fixed-right 280px `#161b22`, energy drawer fixed-bottom (height 0, `transition: height 0.3s`), buttons `#58a6ff` accent, sliders `accent-color: #58a6ff`, `.step-btn` 28×28px, `#help-overlay` fixed inset with `#000000cc`, `.help-content` max-width 600px rounded.
- `js/config.js` — Added `CHARGE_RADIUS_PX = 20` (charge visual radius used by arrow skip zone and charge-renderer).
- `js/renderer/color-map.js` — `_buildViridis()` computes 256 `[r,g,b]` entries from canonical Horner-form polynomial (Shadertoy Viridis fit). `magnitudeToColor(magnitude, maxMag)` → `{r,g,b,alpha}` using `t = log10(1 + mag/maxMag·999) / 3`, alpha = 0.25 + t·0.7.
- `js/renderer/arrow-renderer.js` — `drawArrow()`: log-scale length (MAX=spacing×0.7, MIN=spacing×0.2), Viridis color, shaft+arrowhead. `renderGrid()`: skips grid points within `CHARGE_RADIUS_PX + 4` of any charge center.
- `js/renderer/charge-renderer.js` — `drawCharge()`: circle radius 20px, `#ff6b6b` positive / `#4dabf7` negative, 20px glow, `+/−` symbol. `drawChargeLabel()`: two-line 12px label, flips above if near bottom edge.
- `js/renderer/color-bar.js` — Left side (x=10, y=200, 20×400px). Linear gradient top→bottom (max mag yellow → min mag purple). 5 ticks with `formatField()` labels. Rotated "|E| (V/m)" title.
- `js/renderer/canvas-manager.js` — `CanvasManager` class. `render()` 11-step pipeline (stubs for Phase 4/5 steps). `_drawScaleBar()`, `_drawHelpButton()`, `_drawFullscreenButton()`, `drawScrubCrosshair()` (stub for Phase 5). Imports `FieldEngine` to use `formatField` for color-bar ticks.
- `js/main.js` — Phase 2 visual bootstrap: creates CanvasManager, renders one `+1 μC` charge, posts to grid worker, re-renders with arrows on worker response.
