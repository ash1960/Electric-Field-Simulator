## What Was Built

- `js/interaction/drag-manager.js` — `DragManager` class. Hit-tests charges (r=CHARGE_RADIUS_PX+4) then particles (r=12). `pointerdown/move/up/cancel` listeners. Shift+drag axis-lock (dominant axis determined from total offset since mousedown). Cursor: crosshair/grab/grabbing. Calls `onDragMove(id,type)` on every pointermove during drag — main.js uses this to trigger sub-sampled grid recompute.
- `js/interaction/probe.js` — `Probe` class. Stores last mouse position; `onMouseMove()` updates it and immediately recomputes. `refresh(charges, isDragging)` called from rAF loop to keep probe live even when mouse is still. `clear()` on mouseleave.
- `js/interaction/keyboard.js` — `KeyboardManager`. All 11 shortcuts (Del, C, 1/2/3, F, +/-, S, R, V, Space, ?). Input focus guard: ignores events when `input/select/textarea` is focused.
- `js/interaction/help-overlay.js` — `HelpOverlay`. Builds `#help-overlay` content with 6 sections (Getting Started, Field Probe, Test Particles, Energy Analysis, Keyboard Shortcuts, Units & Scale). Toggle via `.visible` class. Dismiss on background click or Escape.
- `js/interaction/ui-controls.js` — `UIControls`. Dynamically builds `#control-panel` with 9 sections. Slider helper `_makeSlider()` (linear) and `_logSlider()` (log, for mass/speed). Section headers via `.section-header` class. Conditional CHARGE PROPERTIES and PARTICLE PROPERTIES sections. Footer status line with counts and ΣQ.
- `js/interaction/particle-manager.js` — `ParticleManager`. `add()`, `remove()`, `reset(id|'all')`, `clear()`. Particle schema with id, pos, drop_pos, velocity, charge, mass_kg, state, color, trail, simTime.
- `js/main.js` — Full Phase 3 orchestrator. State object (charges, gridResult, gridSpacing, selectedId/Type, showVectors, paused, timeScale, scrubTimestamp). Worker sub-sampling during drag (2× spacing). rAF loop with `probe.refresh()`. Presets: dipole, quadrupole, line. Canvas click handler for ? and fullscreen buttons. All keyboard and UI callbacks wired.
- `js/renderer/canvas-manager.js` — Implemented `_drawProbe()` (was Phase 3 stub): 5-line monospace tooltip box (rounded, dark bg, `#30363d` border) + 24px direction arrow at cursor. `_rrect()` helper for rounded rect. `_sci()` helper for scientific notation with Unicode superscripts. Updated render step 5 to pass `isSelected` to `drawCharge`.
- `js/renderer/charge-renderer.js` — `drawCharge(ctx, charge, isSelected=false)`: white 2px ring + `shadowBlur=12` when selected.
- `css/style.css` — Added `.section-header`, `.slider-value`, `.shortcut-table`, `.help-section`, `.help-content p`, `.help-content dl.shortcut-table` rules.
